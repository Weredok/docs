# $dm
Отправляет сообщение в личные сообщения
### Использование
```php
$dm[id?]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| id |  |  | Нет |  
## Пример(ы)

```javascript
bot.command({
  name: '$dm',
  code: `
$dm[id?]
сообщение`
// Возвращает: В личные сообщения бот отправит сообщение
})
```
