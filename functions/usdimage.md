# $image
Отправляет изображение
### Использование
```php
$image[$userAvatar[$authorID]]
```

### Опции

| Опция | Описание | Тип | Обязательно |
|--------|-------------|------|----------|
| Индекс |  | цифра | Нет |
| Ссылка |  | Ссылка на изображение | Да |
## Пример(ы)

```javascript
bot.command({
  name: '$image',
  code: `
$image[$userAvatar[$authorID]]`
// Возвращает: Изображение на вашу аватарку
})
```
